# 三大模式之页面模式

# 如何使用

在任何页面，使用快捷键在页面中打开Steward界面
快捷键默认为windows: ctrl + j / mac: command + j，可在扩展设置页底部自定义

# 哪些功能

可以使用所有browser插件以及网页插件

# 特点

插件功能最全面

# 注意

由于默认开启的速度优先设置项，页面模式会往每个页面注入一个iframe，并初始化UI界面，消耗一定的内存，对于习惯打开 n 个标签的用户，建议将速度优先设置为关闭
