## 命令
### 执行 javascript 代码
trigger: `> `

#### 操作
- `Input` 输入 javascript 代码
- `Enter` 执行输入的代码，并将结果显示在列表第一行

#### 其它用法
- `$.now` 显示当前时间
- `$.unix` 当前时间的 `unix` 时间戳
- `day` dayjs 函数的 `alias`
- `http` axios 的 `alias`
