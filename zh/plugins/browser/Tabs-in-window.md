## 安装
`spm install tabs in window`

## 命令
### 分离标签页
- trigger: `det `

#### 操作
- `Enter` 将当前标签页移到新窗口打开

### 合并标签页
- trigger: `att `

#### 操作
- `Enter` 将其它窗口的所有标签页合并到当前窗口中
