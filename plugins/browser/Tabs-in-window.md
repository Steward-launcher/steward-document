## Install
`spm install tabs in window`

## Commands
### Detach tabs
- trigger: `det `

#### Operating
- `Enter` Move the current tab to a new window

### Attach tabs
- trigger: `att `

#### Operating
- `Enter` Attach all tabs of other windows into the current window
